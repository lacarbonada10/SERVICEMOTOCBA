# 🏍️ MotoServiceCba - Gestión Integral de Taller

MotoServiceCba es una aplicación web ligera y moderna diseñada para talleres mecánicos de motocicletas. Permite a los usuarios y mecánicos cotizar servicios de mantenimiento preventivo y visualizar opciones de reparaciones pesadas/rectificaciones de manera rápida y visual.

![Versión](https://img.shields.io/badge/version-2.0-orange)
![Tecnologías](https://img.shields.io/badge/tech-HTML5%20|%20TailwindCSS%20|%20JS-blue)

## 🚀 Características

La aplicación se divide en dos módulos principales:

### 1. Cotizador de Service Preventivo
*   **Selección por Cilindrada:** Soporta motores de 50, 70, 90, 110, 150, 200 y 250cc.
*   **Lógica de Precios Automática:**
    *   De 50cc a 110cc: **$60.000**
    *   De 150cc a 250cc: **$80.000**
*   **Checklist de Tareas:** Detalle visual de carburación, transmisión, filtros, aceites y frenos.

### 2. Módulo de Rectificaciones y Reparaciones
*   **Sistema de Persiana (Acordeón):** Información detallada sobre reparaciones complejas.
*   **Categorías Incluidas:**
    *   Tapa de Cilindro
    *   Cilindro
    *   Caja de Velocidades
    *   Cigüeñal
    *   Embrague

## 🛠️ Tecnologías Utilizadas

*   **HTML5 Semántico:** Para una estructura sólida y accesible.
*   **Tailwind CSS:** Framework de estilos utilizado para un diseño responsivo y moderno (Mobile-First).
*   **JavaScript (Vanilla):** Lógica dinámica para el cálculo de costos y manejo de estados de la interfaz sin dependencias pesadas.
*   **FontAwesome:** Iconografía profesional para una mejor experiencia de usuario.
